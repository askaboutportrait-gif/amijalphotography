# Amijal Photography

A responsive photography portfolio website for portraits, weddings, events, and brand sessions.

## Files

- `index.html` contains the page structure and booking form.
- `styles.css` contains the responsive visual design.
- `script.js` powers gallery filtering, mobile navigation, lightbox previews, and the booking form state.
- `thanks.html` confirms a successful booking enquiry.

Open `index.html` in a browser to preview the site locally.

## Booking form

The booking form posts to FormSubmit and sends enquiries to `askaboutportrait@gmail.com`.
The first real submission will send a confirmation email to that inbox; open it and approve the form to activate delivery.
